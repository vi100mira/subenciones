# Subvenciones con IA

Cockpit SaaS para localizar, verificar y preparar subvenciones con evidencia oficial, aislamiento entre entidades y revisión humana.

## Documentación principal

- [Índice documental canónico](docs/documentation-index.md): qué documento responde cada pregunta y cuál es su vigencia.
- [Especificación funcional](docs/product/functional-specification.md): roles, recorridos, estados, reglas y criterios de aceptación.
- [Arquitectura actual del sistema](docs/architecture/arquitectura-actual-del-sistema.md): diagrama, colas, agentes, OCR, datos productivos y límites reales.
- [Modelo de datos](docs/architecture/data-model-reference.md): entidades públicas y privadas, relaciones y aislamiento tenant.
- [Mapa de fuentes](docs/product/source-map.md): cobertura, cadencia, método de captura y riesgos de acceso.
- [Contexto de producto](docs/product/master-context.md): propósito, fronteras de confianza y alcance funcional.
- [Gobierno de datos](docs/product/data-governance-brief.md): clasificación, privacidad y datos prohibidos.
- [Registro de cambios](docs/changelog/): evidencia de implementación y verificaciones.
